# SupraBanking Frontend

Frontend Angular 18 pour la plateforme bancaire SupraBanking.

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

L'appli sera disponible sur http://localhost:4200/

## Build

```bash
npm run build
```

## Structure du Projet

- `src/app/core/` - Services, guards, interceptors
- `src/app/shared/` - Composants et services partagés
- `src/app/features/` - Modules métier (auth, comptes, dashboard)
- `src/app/layout/` - Composants de layout

## Technologies

- Angular 18
- TypeScript 5.4
- RxJS 7.8
- SCSS
